1. VS Code
- File => Add folder to workspace
- Select folder for storing code
- Right click on that folder
- Select "Open in integrated terminal"
- Set Terminal to CMD and set as default
- Type below commands to create Express web app:
npx express-generator --view=hbs
npm install
npm install nodemon -g
npm install mongoose --save
npm install body-parser --save
nodemon

2. Google Chrome
- Enter this URL to open Express web app
localhost:3000
